Omen

Omen was the first widely used level editor for Minecraft, used to edit Classic and Indev levels in 2009 and 2010. Initially it was called Osici's Minecraft Editor, or OME.

The original downloads are still available and are listed here.

  • Omen 0.7.0 2010-02-07
  • Omen 0.6.0 2009-12-02
  • Omen 0.5.0 2009-07-19
  • OME 0.4.1 2009-06-23
  • OME 0.4.0 2009-06-22

Omen requires Java and a Minecraft Classic jar to save Classic levels.
